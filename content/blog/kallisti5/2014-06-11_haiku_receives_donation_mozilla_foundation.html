+++ type = "blog" author = "kallisti5" title = "Haiku receives donation from Mozilla Foundation" date = "2014-06-11T19:09:13.000Z" tags = [] +++ The Mozilla Foundation has generously donated 15 gently used Intel Mac Minis to Haiku, Inc. to be used as infrastructure, development, and build systems.

These systems are planned to be deployed as updated buildbot slaves, package builders, and used to better support Intel Mac Mini hardware.
