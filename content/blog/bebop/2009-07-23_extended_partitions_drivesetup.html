+++ type = "blog" author = "bebop" title = "Extended Partitions with DriveSetup" date = "2009-07-23T09:41:48.000Z" tags = ["DriveSetup", "disk_device", "gsoc 2009"] +++ DriveSetup with extended partitions I few days ago a patch went into the tree that cleaned up most of the remaining issues with creating primary partitions. So I started working on getting extended partition support in DriveSetup. This has been an interesting task as the extended partition support is not nearly as complete as primary partition support. To make a long story short, I have been able to create an extended partition and logical partitions within. This code is just a hack at the moment and not fit for the public but hopefully in the coming days the code can be cleaned up and the bugs can be fixed and we will have extended partitions!
